## Required !
  - Node v.16.14 or higher !
  - Yarn Package Manager.

## Available Commands
  - Slash Commands
    - Audio
      - Play : play song.
      - Skip : skip the current song.
      - Queue : list all track and upcoming track.
      - Clear : clear upcoming track from queue.
    - Common
      - Covid : check covid stats.
      - Info : get bot info.
  - Message Commands
    - no commands available now
